# CEYLON X 🚀
### 2026 Adaptive Super-Intelligence AI Agent

**Ceylon X** is the world's most advanced autonomous AI agent for your terminal. Built for the 2026 era, it combines elite software engineering, cybersecurity, business strategy, and real-time market analysis in a single, high-performance interface.

---

## 🌟 Super-Intelligence Features (v1.3.0)

- 🤖 **Universal Multi-Provider Engine**: Support for 13+ top AI providers (Gemini, Anthropic, OpenAI, OpenRouter, Groq, etc.) with intelligent auto-switching.
- 💻 **Elite Engineering Mode (/code)**: Advanced 5-step autonomous flow (Analysis -> Research -> Plan -> Code -> QA) with self-healing logic for professional development.
- 🔐 **Secure API Vault**: Machine-level encrypted local storage (`aes-256-cbc`) to manage all your keys safely without re-entering them.
- 🖼️ **Universal Image Engine**: High-detail visual generation powered by **Ghostbot AI** (11+ models) and Pollinations, featuring auto-polling and premium quality.
- 📈 **Trading Signals**: Real-time market analysis for Crypto/Forex with structured [BUY/SELL] signals.
- 📊 **Business Intelligence**: Automated professionally styled Excel reports and PowerPoint presentations.
- 🎙️ **Voice Studio**: Instant, free Text-to-Speech (TTS) for any text in clear, high-quality audio.
- 🔍 **SEO Auditor**: In-depth website scraping and technical SEO auditing.

## 🚀 Installation

### Option 1: Global Install (Fastest)
Install globally via NPM:
```bash
npm install -g ceylonx
```

### Option 2: Local Install (For Developers)
Clone the repository and build the protected production version:
```bash
git clone https://github.com/your-repo/ceylonx.git
cd ceylonx
npm install
npm run build
npm install -g .
```

## ⚡ Quick Start

1. **Start Ceylon X**:
   ```bash
   ceylonx
   ```

2. **Setup Your Vault**:
   Run the vault command to securely store your API keys once:
   ```bash
   /vault
   ```

3. **Autonomous Coding**:
   Use elite mode for complex tasks:
   ```bash
   /code "Build a full-stack Next.js dashboard with authentication"
   ```

## 🛡️ Security & Deployment

Ceylon X uses **Advanced Identity Protection**:
- **Source Obfuscation**: The production code is cryptographically obfuscated to protect internal logic and developer keys.
- **Local Encryption**: All user API keys are encrypted using your machine's unique hardware ID as a salt.
- **Manual Guards**: Ceylon X will NEVER execute terminal commands, delete files, or commit to Git without your explicit permission.
